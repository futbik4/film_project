# Django Film Management App

Приложение для управления информацией о фильмах с возможностью импорта/экспорта в JSON формате.

## Работающее приложение
**Доступно по адресу:** https://futbik4.pythonanywhere.com/

## Функциональность

- Добавление фильмов с выбором места сохранения (БД/JSON)
- Проверка на дубликаты при сохранении в БД
- Просмотр данных из БД или файлов
- AJAX-поиск по фильмам
- Редактирование и удаление записей
- Импорт/экспорт данных

## Установка и запуск

1. **Клонируйте репозиторий:**
```
git clone <https://github.com/futbik4/film_project>
cd film_project
```
2. **Создайте виртуальное окружение:**
```
py -m venv venv
```
3. **Активируйте виртуальное окружение:**
```
venv\Scripts\activate
```
4. **Установите зависимости:**
```
pip install -r requirements.txt
```
6. **Выполнить миграции**
```
py manage.py makemigrations
py manage.py migrate
```
7. **Запустите сервер:**
```
py manage.py runserver
```
8. **Откройте в браузере:**
<http://127.0.0.1:8000/>

## Технологии
