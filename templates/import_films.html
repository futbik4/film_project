Импорт фильмов из файла

Поддерживаемые форматы: JSON, XML

{% csrf_token %}
Файл должен быть в формате JSON или XML с данными о фильмах
Пример формата JSON:
[
    {
        "title": "Название фильма",
        "director": "Режиссер", 
        "year": 2020,
        "genre": "Жанр",
        "rating": 8.5,
        "description": "Описание"
    }
]
Пример формата XML:
<films>
    <film>
        <title>Название фильма</title>
        <director>Режиссер</director>
        <year>2020</year>
        <genre>Жанр</genre>
        <rating>8.5</rating>
        <description>Описание</description>
    </film>
</films>
{% endblock %}
